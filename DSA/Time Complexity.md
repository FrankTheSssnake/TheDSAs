- Time Complexity =/= Time Taken
- We use Big O Notation to measure TC
  `O(...)`

## Big O Notation

- TC is complicated using the worst case scenario
- Avoid constants
- Avoid lower values

Note:
- Big O Notation: "Upper Bound"
- Theta Notation: "Average"
- Omega Notation: "Lower Bound"
